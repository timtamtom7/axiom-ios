# AxiomIOSMac — R12: Therapist Integration & Guided Exercises

## Theme
Professional therapeutic integration, guided CBT exercises, and community support features.

## Features

### R12.1 — Therapist Partnership Mode
- **Provider Code Entry:** Therapists can generate share codes for clients
- **Progress Sharing:** Client shares belief progress, evidence counts, and trend data
- **Annotation System:** Therapist adds private notes to client beliefs
- **Assignment Feature:** Therapist assigns specific exercises to clients
- **Session Prep Reports:** Auto-generate summary reports before therapy sessions
- **Two-way Messaging:** Secure in-app communication channel

### R12.2 — Guided CBT Exercises Library
- **Exercise Types:**
  - Decatastrophizing worksheets
  - Probability re-estimation exercises
  - Evidence weighing activities
  - Behavioral activation suggestions
  - Exposure ladder builder
- **Interactive Workflow:** Step-by-step guided completion
- **Exercise History:** Track which exercises were completed
- **Custom Exercise Creation:** Therapists create custom exercises for clients
- **Reminders:** Scheduled exercise reminders based on therapist recommendations

### R12.3 — Guided Breathing & Grounding
- **4-7-8 Breathing:** Built-in breathing guide with visual cue
- **5-4-3-2-1 Grounding:** Sensory grounding exercise for anxiety
- **Body Scan Prompts:** Progressive relaxation guidance
- **Crisis Resources:** Quick access to crisis helpline information
- **Mood Check-in Integration:** Trigger exercises based on mood entries

### R12.4 — Community Support Enhancements
- **Anonymous Debates:** Structured debate format with evidence requirements
- **Wisdom Library:** Moderated collection of community-shared insights
- **Accountability Partners:** Optional pairing with community members
- **Moderation System:** Report and flag inappropriate content
- **Reputation System:** Earn trust badges through helpful contributions

### R12.5 — Professional Features
- **HIPAA Mode:** Optional encrypted sync with therapist
- **Multi-client Dashboard:** Therapists manage multiple clients (future subscription tier)
- **Export to EHR:** Export session notes in compatible formats

## Technical
- **Encryption:** End-to-end encryption for therapist-client communication
- **Privacy-first:** Minimal data sharing; client controls what therapists see
- **Offline-first:** All exercises work without internet

## Design Notes
- Warm, supportive aesthetic — this is clinical but not cold
- Clear distinction between self-guided and therapist-assisted features
- Professional credibility without intimidating presentation
